### Folders Structure

- Data are prepared as raw data or engineered data.
- Download the dataset files [Google Drive](https://example.com) | [Zenodo](https://example.com) | [Google Drive](https://example.com).
- The directory structure is as follows:
```
data
└── raw
   ├── apps.csv
   ├── reviews.csv
   └── categories/
└── engineered
   ├── apps.csv
   ├── reviews.csv
   └── vocabulary.csv
```
